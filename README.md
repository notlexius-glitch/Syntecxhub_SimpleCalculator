# Syntecxhub Simple Calculator

A simple command-line calculator built in Python as part of the **Syntecxhub Internship**.

## 📌 Features

* Addition
* Subtraction
* Multiplication
* Division
* User-friendly menu
* Handles invalid input
* Prevents division by zero

## 🛠️ Technologies Used

* Python 3

## 📂 Project Structure

```
Syntecxhub_SimpleCalculator/
├── calculator.py
├── README.md
└── .gitignore
```

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Clone the repository:

```bash
git clone https://github.com/notlexius-glitch/Syntecxhub_SimpleCalculator.git
```

3. Navigate to the project folder:

```bash
cd Syntecxhub_SimpleCalculator
```

4. Run the program:

```bash
python calculator.py
```

> If your system uses `python3`, run:

```bash
python3 calculator.py
```
